# Pac Man

A Pac Man game powered by pygame. (XMUM 2102 G0234 Project)

## Requirements

```
pygame==2.0.1
numpy==1.20.1
scipy==1.6.2
pillow==8.1.2
```

You can use `pip` to install required packages:
```sh
python -m pip install -r requirements.txt
```

## Play

```sh
cd pacman
python main.py
```
